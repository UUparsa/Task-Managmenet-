# Task-Managmenet-
This  project is made for Soft_Skill lesson 
برنامه مدیریت کار ها (Task Manager)

این پروژه یک برنامه برای مدیریت تسک است که با استفاده از HTML، CSS و JavaScript پیاده‌سازی شده است.  
هدف اصلی این پروژه تمرین همکاری تیمی، برنامه‌ریزی با Trello و کار کردن  با GitHub است.

هدف این پروژه:
- تمرین برنامه‌ریزی مشارکتی
- تقسیم وظایف بین اعضای 'v,i
- مدیریت تعارض‌ها و تصمیم‌گیری 
- استفاده صحیح از GitHub (Branch، Pull Request و Review)

می‌باشد.

 امکانات برنامه:

- افزودن تسک جدید (عنوان و توضیح)
- نمایش لیست تسک‌ها
- تغییر وضعیت تسک (To Do / In Progress / Done)
- حذف تسک با پیام تأیید (Confirmation)
- رابط کاربری ساده و قابل استفاده

 تکنولوژی‌های استفاده‌شده:

 HTML
CSS
JvavScript
 Trello
 GitHub (برای کنترل نسخه و همکاری)

 نحوه اجرای پروژه

   2. وارد پوشه پروژه شوید:

 همکاری تیمی

این پروژه توسط یک تیم دو نفره انجام شده است.  
نقش‌ها در ابتدای پروژه مذاکره و در Trello ثبت شده‌اند.

- یک نفر مسئول ساختار HTML و CSS پروژه
- یک نفر مسئول  JavaScript و بررسی کیفیت (QA)

 قوانین گردش کار (Workflow)

- شاخه main محافظت شده است
- هیچ Commit مستقیمی روی main انجام نمی‌شود
- تمام تغییرات از طریق Feature Branch انجام می‌شوند
- هر تغییر باید Pull Request داشته باشد
- حداقل یک Reviewer برای هر PR الزامی است

 توضیح پایانی

این پروژه به‌صورت آگاهانه ساده نگه داشته شده است تا تمرکز اصلی روی:
- برنامه‌ریزی
- ارتباط تیمی
- حل تعارض
- همکاری مؤثر

باشد.
